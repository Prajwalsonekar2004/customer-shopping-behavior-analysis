#  Customer Shopping Behavior Analysis

##  Overview
This project explores **customer shopping behavior** using **Python, PostgreSQL, and Power BI**.  
It focuses on analyzing purchase trends, spending patterns, and product preferences to generate actionable insights for retail decision-making.

---

##  Objectives
- Understand customer purchase trends and behavior  
- Identify top-selling products and peak purchase times  
- Analyze revenue distribution across categories  
- Build an interactive Power BI dashboard for business insights  

---

##  Tools & Technologies
- **Python:** Data cleaning, exploration, and visualization  
- **PostgreSQL:** Data storage, querying, and transformation  
- **Power BI:** Dashboard creation and storytelling  

---

##  Key Steps
1. **Data Collection:** Imported customer shopping dataset into PostgreSQL  
2. **Data Cleaning:** Handled missing values and formatted data using Python (Pandas)  
3. **Data Analysis:** Executed SQL queries for aggregation and trend analysis  
4. **Visualization:** Created interactive Power BI dashboard with KPIs and visuals  

---

##  Insights & Results
- Identified **top 10 most purchased products** and **high-revenue categories**  
- Discovered **seasonal purchase patterns** and **customer preferences**  
- Improved understanding of **spending behavior** using Power BI visuals  

---

##  Business Recommendations
- **Boost Subscriptions:** Promote exclusive benefits for subscribers.  
- **Customer Loyalty Programs:** Reward repeat buyers to move them into the “Loyal” segment.  
- **Review Discount Policy:** Balance sales boosts with margin control.  
- **Product Positioning:** Highlight top-rated and best-selling products in campaigns.  
- **Targeted Marketing:** Focus efforts on high-revenue age groups and express-shipping users.
